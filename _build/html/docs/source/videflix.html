class videflix.authentication.CookieTokenAuthentication[source]¶

Bases: BaseAuthentication

Custom authentication class that authenticates users via a token stored in cookies.

- authenticate(request)

Attempts to retrieve the ‘auth_token’ from the request cookies. If a token is found, it validates the token and returns the associated user. If the token is invalid or does not exist, it raises an AuthenticationFailed exception or returns None.

authenticate(request)[source]¶

Authenticates the user based on the token in the ‘auth_token’ cookie.

Parameters:

request – The HTTP request containing the cookie.

Returns:

A tuple of (user, token) if the token is valid. None if no token is provided or the token is invalid.

Raises:

AuthenticationFailed – If the token is invalid.
